Website Conversion

Existing Static HTML converted to Responsive HTML using Bootstrap and Javascript.

  • This is a conversion, not a re-design. I re-created the existing static website's lay-out pixel by pixel (or a few pixel apart).
  • No template was used, I re-wrote the code from scratch using bootstrap and Jquery.
  • Navbar menu and footer is in separate HTML, it was loaded in each page so changing them will be easy, you don't have to change it in each html page.
  • The navbar menu in the existing website is not aligned, so i tweaked it a little bit so it lined up.
  • Box shadow in the existing website is an image file, when i tried to use it for the Responsive website, the image doesn't shrink properly for the small screen so i just used plain CSS (box-shadow).
